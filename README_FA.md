# وب‌اپ مدیریت حسابداری چندشرکتی ۱۴۰۵

این پروژه یک وب‌اپ سبک PHP + MySQL برای هاست‌های cPanel است. هدف آن تبدیل فایل اکسل مدیریت حسابداری چندشرکتی به یک سیستم تحت وب دائمی است که داده‌ها را از دیتابیس می‌خواند و در دیتابیس ذخیره می‌کند.

## امکانات اصلی

- لاگین با ایمیل/رمز عبور
- ورود و ثبت‌نام با گوگل OAuth 2.0
- داشبورد مدیریتی با KPIهای پویا
- مدیریت ۵ شرکت پیش‌فرض:
  - موسسه شهر کتاب مرکزی
  - فروشگاه شهر کتاب مرکزی
  - رستگار صنعت پارس
  - کیوان الکترونیک ایرانیان
  - کیهان توسعه البرز
- مدیریت کارها و سررسیدهای ۱۴۰۵
- تقویم سررسیدهای آینده
- پیگیری درخواست‌ها
- مغایرت‌گیری بانکی
- بانک اطلاعات سامانه‌ها
- ثبت خطاها و تجربیات حسابداری
- ایمپورت از فایل xlsx
- خروجی xlsx از دیتابیس
- تنظیمات SMTP Gmail برای ارسال ایمیل
- تنظیمات قاصدک برای ارسال پیامک
- Cron Job برای ارسال خودکار یادآوری‌ها

## تکنولوژی

- PHP 8+
- MySQL 8
- Apache/cPanel
- بدون نیاز به Composer و Node.js

## نکته امنیتی

فایل `app/config.php` بعد از نصب ساخته می‌شود و شامل اطلاعات دیتابیس است. `.htaccess` دسترسی مستقیم به فایل‌های حساس را محدود می‌کند. با این حال بهتر است کل پروژه را فقط داخل ساب‌دامین خصوصی قرار دهید و از رمز قوی استفاده کنید.

## فایل نمونه اکسل

داخل مسیر زیر یک نمونه از فایل اکسل اولیه قرار داده شده است:

`sample_import/Accounting_Manager_5Companies_1405.xlsx`

از بخش «اکسل/پشتیبان» داخل وب‌اپ می‌توانید این فایل را ایمپورت کنید یا از دیتابیس خروجی اکسل بگیرید.
